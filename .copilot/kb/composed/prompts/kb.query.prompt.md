# 查詢知識庫

你正在從知識庫中搜尋答案。

## 任務
根據使用者的問題，找到最相關的知識並回傳清晰、附來源的答案。

此入口是**嚴格以 KB 為依據**的查詢模式。若 KB 沒有足夠證據，必須明確說明，不得自行擴充超出文件內容。若使用者需要在 KB 背景上進一步得到建議，改用 `/kb.advise`。

## 自動處理事項
- 導覽索引以找到相關文章
- 只閱讀最少量的必要文件
- 整合多個來源的資訊
- 回報信心度與知識缺口

使用者無需了解索引結構或規劃搜尋路徑。

## 輸出要求
回傳：
1. 問題的答案
2. 使用的來源（檔案路徑）
3. 信心度（高 / 中 / 低）
4. 發現的知識缺口（若有）
